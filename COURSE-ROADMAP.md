# Course Roadmap

The diploma is divided into four progressive phases.

---

# Phase 1 — Foundations

Goal:

Understand Microservices Architecture and build the platform foundation.

Sessions

00 Course Kickoff

01 Architecture & Spring Cloud

02 API Gateway Routing

03 Gateway Security

04 Resilience

05 Configuration Management

06 Service Communication

07 Event Driven Architecture

08 Phase Review

---

# Phase 2 — Business Services

Goal:

Build business microservices using DDD and asynchronous communication.

Includes:

Order Service

Inventory Service

Payment Service

Notification Service

Saga Pattern

Kafka

---

# Phase 3 — Cloud Native

Goal

Deploy production-ready systems.

Topics include:

Docker

Kubernetes

CI/CD

GitHub Actions

Observability

Distributed Tracing

---

# Phase 4 — Enterprise

Goal

Production-grade architecture.

Topics include:

Security

OAuth2

Performance

Scaling

Caching

Advanced Testing

Capstone

---

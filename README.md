# Microservices Professional Diploma

> A comprehensive, project-based Microservices training program using Spring Boot 3, Spring Cloud, Docker, Kafka, Kubernetes, GitHub Actions, and modern cloud-native architecture.

---

## About This Course

The **Microservices Professional Diploma** is an intensive hands-on program designed for software engineers who want to build production-ready distributed systems using modern Java technologies.

Unlike traditional courses that focus on isolated examples, this diploma follows a **Continuous Capstone Project** approach. Throughout the program, trainees build a complete Enterprise E-Commerce Platform incrementally across 29 sessions.

By the end of the course, participants will have designed, implemented, tested, deployed, and monitored a complete cloud-native microservices platform.

---

## What You Will Build

Throughout the diploma, trainees will develop an enterprise-grade E-Commerce platform including:

- API Gateway
- Service Discovery (Eureka)
- Centralized Configuration
- Product Service
- Inventory Service
- Order Service
- Payment Service
- Notification Service
- Kafka Event Streaming
- Saga Pattern
- Distributed Tracing
- Kubernetes Deployment
- CI/CD Pipelines

---

## Learning Methodology

This course follows a structured learning cycle for every session.

```
Concept
    ↓
Architecture Discussion
    ↓
Live Coding
    ↓
Hands-on Lab
    ↓
Knowledge Check
    ↓
Homework
    ↓
Checkpoint Commit
```

Learning is project-based rather than slide-based.
